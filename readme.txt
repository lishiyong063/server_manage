#介绍 ：一共3个文件：main.py  config.py  conf 文件夹
main.py 主程序,config.py 一些配置文件，conf 是存储主机信息的配置文件

运行程序：python main.py 123 #123是密码 因为不用数据库 密码写程序里面了

 一.先添加一个分组：选1 服务组管理 ,然后3 添加分组，输入新加分组的名字 web,会提示是否添加主机enter 添加主机. 这里主机信息 请填写正确的信息
 
 填写完成 后 会提示 确认填写的信息。 输入y， 问是否继续添加 n 返回上级。可以查看分组信息 
 选2 。上面会显示刚才添加的组名信息。可以修改删除单独添加主机信息（可以多添加几条）
 
 删除一个组的时候 其实是将组配置文件名改了 在conf目录下面有备份 可以改回原来的状态。
 
 
 修改分组包括：
         1 - 删除主机 

         2 - 修改组名 

         3 - 修改组员信息   
 
 二.exit 返回上级菜单 选2 执行批量操作 提示 选择 组名  web   确认  再ls 看是否成功。当然可以试着执行一些其他的命令
 
 三.exit 返回上级菜单 文件分发  ，选择操作的组名 web. 问你要分发的文件名、ls可以查看当前目录下的文件 有哪些选择一个分发
 确认后 如果没有报错 说明分发成功、可以选择继续分发文件  
 
 四.exit 返回上级菜单后 选择4 退出系统。
